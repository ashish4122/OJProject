# 🧠 Codex - Online Judge System

Codex is a MERN stack-based online judge platform where users can solve coding problems, submit code in multiple languages, and get instant verdicts in a secure, isolated environment.

---

## 🚀 Features

### 👤 Users
- Register/Login
- Browse & solve coding problems
- Code editor with multi-language support (C++, Java, Python)
- Instant verdicts: Accepted, Wrong Answer, TLE, RE
- View submission history

### 🔐 Admins
- Add/Edit/Delete problems with constraints & test cases
- View all user submissions

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Axios, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Execution Engine:** Docker (Isolated runtime)
- **Deployment:** AWS / Render / Vercel
